## Características:
Herencia y Polimorfismo: Implementa una clase base Recurso y subclases Libro y Revista para manejar diferentes tipos de materiales.

Gestión de Estados: Controla la disponibilidad de los ítems para evitar préstamos duplicados.

Encapsulamiento: Protege los datos internos de los libros y usuarios, exponiendo solo lo necesario a través de métodos.

Interfaz Profesional: Desarrollado con wxPython utilizando tablas (ListCtrl) para visualizar el catálogo de forma clara.
